\section{Introduction}
\label{sec:intro}

This document demonstrates \important{advanced compiler features} including:
\begin{itemize}
    \item Symbol table management
    \item Cross-reference resolution
    \item Macro expansion (with and without parameters)
    \item Automatic numbering
\end{itemize}

As we'll see in \ref{sec:features}, these features greatly enhance the compiler.

\section{Advanced Features}
\label{sec:features}

\subsection{Symbol Table}
\label{subsec:symtab}

The \highlight{symbol table} tracks:
\begin{enumerate}
    \item Labels and their locations
    \item Section and equation numbers
    \item User-defined macros
    \item Cross-references
\end{enumerate}

For example, this is Section \ref{sec:features}, subsection \ref{subsec:symtab}.

\subsection{Macro Expansion}

We defined three custom macros:
\begin{itemize}
    \item \texttt{\\important\{text\}} - creates \important{important text}
    \item \texttt{\\highlight\{text\}} - creates \highlight{highlighted text}  
    \item \texttt{\\note\{title\}\{text\}} - creates \note{Example}{a formatted note}
\end{itemize}
